Govhoo API

A single POST endpoint that proxies the USASpending.gov federal contracts database. Query award data, agency spending, vendor concentration, subcontracts, and spending trends — without dealing with CORS or USASpending's raw request format directly.

No API key required. The proxy is publicly accessible. All data is sourced directly from USASpending.gov — the official federal spending database maintained by the U.S. Treasury. Data reflects contracts reported by federal agencies.
Base URL

All requests are POST with Content-Type: application/json. No auth headers needed.

endpoint
https://76unm1rlj0.execute-api.us-east-1.amazonaws.com/default/grumpy2

Every request body must include request_type: "usaspending_proxy", an endpoint path, and a payload object forwarded to USASpending.

Search Awards
POST · /search/spending_by_award/

Search federal contract awards by keyword, agency, NAICS, PSC, or date range. Returns up to 100 results per page sorted by amount.

FieldTypeDescription
filters.time_periodarrayrequiredArray of {"start_date","end_date"} in YYYY-MM-DD.
filters.award_type_codesarrayrequired["A","B","C","D"] for contracts only.
filters.keywordsarrayoptional1–2 word terms matched against descriptions.
filters.agenciesarrayoptional[{"type":"awarding","tier":"toptier","name":"..."}]
filters.naics_codesobjectoptional{"require":["541512"]} — 6-digit codes.
filters.psc_codesarrayoptional4-character PSC codes: ["D307"]
limitnumberoptional1–100. Default 100.
sort / orderstringoptional"Award Amount" / "desc"
javascript
const res = await fetch('https://76unm1rlj0.execute-api.us-east-1.amazonaws.com/default/grumpy2', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    request_type: 'usaspending_proxy',
    endpoint: '/search/spending_by_award/',
    payload: {
      filters: {
        time_period: [{ start_date: '2024-04-01', end_date: '2025-04-01' }],
        award_type_codes: ['A','B','C','D'],
        keywords: ['zero trust']
      },
      fields: ['Award ID','Recipient Name','Awarding Agency',
               'Award Amount','Start Date','End Date','Description'],
      limit: 10, sort: 'Award Amount', order: 'desc'
    }
  })
});
const { results } = await res.json();
Top Agencies by Spend
POST · /search/spending_by_category/awarding_agency

Aggregate spend grouped by awarding agency. Returns a ranked list useful for market sizing and territory planning.

javascript
const res = await fetch(PROXY, {
  method: 'POST', headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    request_type: 'usaspending_proxy',
    endpoint: '/search/spending_by_category/awarding_agency',
    payload: {
      category: 'awarding_agency',
      filters: {
        time_period: [{ start_date: '2024-04-01', end_date: '2025-04-01' }],
        award_type_codes: ['A','B','C','D'],
        keywords: ['cloud']
      },
      limit: 10
    }
  })
});
const { results } = await res.json();
// [{ name: "Dept of Defense", aggregated_amount: 2100000000 }, ...]

Spending Over Time
POST · /search/spending_over_time/

Obligation totals grouped by fiscal year, quarter, or month. Use this to show market growth trends or seasonal buying patterns.

javascript
const res = await fetch(PROXY, {
  method: 'POST', headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    request_type: 'usaspending_proxy',
    endpoint: '/search/spending_over_time/',
    payload: {
      group: 'fiscal_year', // or 'quarter' or 'month'
      filters: {
        time_period: [{ start_date: '2020-10-01', end_date: '2025-04-01' }],
        award_type_codes: ['A','B','C','D'],
        keywords: ['artificial intelligence']
      }
    }
  })
});
const { results } = await res.json();
// [{ time_period: { fiscal_year: "2024" }, aggregated_amount: 4200000000 }, ...]
Subawards
POST · /search/spending_by_award/ (subawards: true)

Search subcontracts — awards made by primes to smaller vendors. Add subawards: true to the payload. Useful for identifying teaming and channel partner activity.

javascript
const res = await fetch(PROXY, {
  method: 'POST', headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    request_type: 'usaspending_proxy',
    endpoint: '/search/spending_by_award/',
    payload: {
      subawards: true,
      filters: {
        time_period: [{ start_date: '2024-04-01', end_date: '2025-04-01' }],
        award_type_codes: ['A','B','C','D'],
        keywords: ['cloud']
      },
      fields: ['Sub-Award ID','Sub-Awardee Name','Sub-Award Amount',
               'Prime Recipient Name','Awarding Agency'],
      limit: 25, sort: 'Sub-Award Amount', order: 'desc'
    }
  })
});
const { results } = await res.json();
// [{ "Sub-Awardee Name": "ACME LLC", "Prime Recipient Name": "LEIDOS", ... }]
Company Name Lookup
POST · /autocomplete/recipient/

Find the exact legal name a company uses in USASpending. Brand names often differ from registered entity names — this resolves them. Also available: /autocomplete/awarding_agency/ for agency name lookup.

javascript
const res = await fetch(PROXY, {
  method: 'POST', headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    request_type: 'usaspending_proxy',
    endpoint: '/autocomplete/recipient/',
    payload: { search_text: 'Primer', limit: 5 }
  })
});
const { results } = await res.json();
// [{ recipient_name: "PRIMER FEDERAL" }, { recipient_name: "PRIMER TECHNOLOGIES" }]

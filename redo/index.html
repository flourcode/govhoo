How it works

Type a keyword, company, agency, or NAICS code. Use AND to intersect (e.g. DHS AND Splunk), commas for OR (Cloud, Cyber). Keep terms exact, this is a live database, not a search engine. In seconds you'll see who's buying, who's winning, and what's expiring. Tap Ask Arti for a pre-meeting briefing based on the data.

About the data

Every search queries the USASpending.gov API v2 in real time, no cache, no stale snapshots. Dollar figures reflect full lifetime obligated value of multi-year contracts. Only active contracts with funding actions in the past 12 months are shown. Searches are routed through Fedhoo's proxy to USASpending and are not stored or logged. Fedhoo is not an opportunity tracker or RFP database. It's market intelligence so you can decide where to focus.
